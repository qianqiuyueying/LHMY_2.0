# 端侧系统字符原型索引（ASCII Wireframes）

本目录用于存放“陆合铭云健康服务平台”各端侧系统的**简易字符原型图**（低保真），用于澄清页面结构、信息层级与主导航，不代表最终视觉稿。

## 原型文档列表

- `website.md`：企业官网（品牌展示与对外信息入口）
- `admin.md`：平台运营后台（admin）
- `dealer.md`：经销商后台（dealer）
- `provider.md`：健康场所/服务提供方后台
- `mini-program.md`：小程序端（健康主题电商平台：三业务线统一入口与使用端）
- `h5.md`：H5 端（营销页 + 健行天下高端服务卡唯一购买入口）
