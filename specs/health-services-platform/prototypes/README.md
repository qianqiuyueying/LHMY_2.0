字体与排版：标题加粗、正文常规，数字与价格突出；模块间距大于元素间距；优先卡片分组；每屏 1 个主行动按钮。

图形与装饰：仅使用线性图标、圆点网格、弧形/波纹几何块、渐变色带；避免复杂纹理与拟物。
交互：反馈及时（点击态/禁用态/加载态），移动端底部 CTA 固定，列表支持骨架屏与空态引导。
负面提示词（强制避免）：
