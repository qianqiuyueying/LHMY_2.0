# H5 端（营销页 + 健行天下高端服务卡唯一购买入口）—字符原型

> 说明：H5 承载营销落地与“高端服务卡”购买闭环；购买成功后，权益在小程序端可见并使用，小程序端不提供购买页。

## 1. 页面原型

### 1.1 营销落地页（Landing）

```
┌─────────────────────────────────────┐
│ 顶部：LOGO / 活动标题                 │
├─────────────────────────────────────┤
│ 主视觉Banner（卖点/权益/保障）         │
├─────────────────────────────────────┤
│ 高端服务卡介绍                        │
│ - 服务范围/区域级别/等级说明           │
│ - 包含服务类别 × 次数                 │
│ - 适用说明/使用方式（小程序使用）      │
├─────────────────────────────────────┤
│ 价格区：                              │
│ - 售价：¥xxxx                         │
│ - 说明：购买后权益在小程序使用         │
├─────────────────────────────────────┤
│ 常见问题/用户须知/条款                 │
├─────────────────────────────────────┤
│ 底部固定CTA： [立即购买] [咨询]         │
└─────────────────────────────────────┘
```

### 1.2 购买页（下单确认）

```
┌─────────────────────────────────────┐
│ 标题：确认购买                         │
├─────────────────────────────────────┤
│ 商品：高端服务卡（名称/规格）           │
│ 数量：[-] 1 [+]                        │
│ 价格：¥xxxx                            │
├─────────────────────────────────────┤
│ 购买人信息：手机号[________] [获取验证码]│
│ 验证码：[____]                         │
├─────────────────────────────────────┤
│ 协议： [✓] 我已阅读并同意《服务协议》   │
├─────────────────────────────────────┤
│ [提交订单并支付]                       │
└─────────────────────────────────────┘
```

说明（最小可执行口径）：
- “获取验证码”用于校验手机号归属（登录/下单前置），验证码有有效期与频控；频控或校验失败时，提示原因并阻断提交支付。

### 1.3 支付结果页（成功/失败）

成功：
```
┌─────────────────────────────────────┐
│ ✅ 支付成功                            │
├─────────────────────────────────────┤
│ 提示：权益已发放，请前往小程序查看并使用 │
│ [打开小程序查看权益]                   │
│ [返回首页]                             │
└─────────────────────────────────────┘
```

失败：
```
┌─────────────────────────────────────┐
│ ❌ 支付失败                            │
│ 原因：xxxxxxxx                         │
│ [重新支付] [联系客服]                  │
└─────────────────────────────────────┘
```

## 2. 经销商参数（仅原型占位）

```
URL 示例：/h5/buy?dealerId=DEALER_ID&ts=1700000000&nonce=RANDOM&sign=HMAC_SHA256_HEX

参数说明（最小可执行）：
- dealerId：归属经销商ID
- ts：秒级时间戳（有效期 10 分钟）
- nonce：随机串（防重放/便于审计）
- sign：签名（HMAC-SHA256，canonical 串为 `dealerId={dealerId}&ts={ts}&nonce={nonce}`）

页面可显示（可选）：归属经销商/活动批次（用于内部对账，不影响用户主流程）
```
