# 企业官网（品牌展示与对外信息入口）—字符原型

## 1. 信息架构 / 主导航

```
顶部导航： [LOGO]  首页  业务线  场所/服务  内容中心  关于我们  联系我们  [进入小程序] [H5购买高端服务卡]
```

## 3. 关键跳转规则（与其他端的关系）

- 官网可提供：**进入小程序**、**跳转 H5 购买高端服务卡** 的入口（仅导流，不承载交易闭环）。
