```
┌──────────────────────────────────────────────────────────────────────────┐
│ 顶栏：LOGO | 经销商后台 | 当前经销商[▼] | 通知[ ] | 账号[dealer▼]            │
├───────────────┬──────────────────────────────────────────────────────────┤
│ 侧栏菜单        │ 主内容区                                                     │
```

## 2. 关键页面原型

### 2.1 仪表盘（转化与归属概览）

```
┌──────────────────────────────────────────────────────────────────────────┐
│ 今日：访问UV | 下单数 | 支付数 | 支付金额 | 退款数                           │
├──────────────────────────────────────────────────────────────────────────┤
│ 图表：链接访问趋势(7/30天)   支付趋势   退款趋势                             │
├──────────────────────────────────────────────────────────────────────────┤
│ 待处理：结算对账( n )  异常订单( n )                                          │
└──────────────────────────────────────────────────────────────────────────┘
```

### 2.2 链接/参数管理（H5 入口生成）

> 对应设计文档：H5 链接携带经销商参数、参数解析与防篡改（正确性属性 5）。

```
Tab：[链接列表] [生成新链接]

[生成新链接]
┌──────────────────────────────────────────────────────────────────────────┐
│ 选择商品：高端服务卡[▼]   活动/批次[____]                                    │
│ 归属经销商：自动(当前) / 选择下级[▼]                                         │
│ 有效期：起[日期] 止[日期]                                                    │
│ 备注： [__________________________]                                          │
│ [生成链接]                                                                  │
├──────────────────────────────────────────────────────────────────────────┤
│ 生成结果：                                                                  │
│  - H5 URL： https://.../h5/buy?dealerId=...&ts=...&nonce=...&sign=...        │
│  - 二维码： [二维码占位]                                                     │
│  - 复制按钮：[复制链接] [下载二维码]                                         │
└──────────────────────────────────────────────────────────────────────────┘

[链接列表]
筛选：高端服务卡[▼] 状态[启用/停用/过期] 创建时间[起-止] 关键字[____] [查询]
┌──────────────────────────────────────────────────────────────────────────┐
│ 链接ID | 商品 | 活动/批次 | 状态 | 有效期(起-止) | 创建时间 | 访问UV | 支付数 | 操作[复制/停用/统计] │
└──────────────────────────────────────────────────────────────────────────┘
```

签名口径（最小可执行，便于前后端联调/验收）：
- 参数：dealerId、ts（秒级时间戳）、nonce（随机串）、sign（HMAC-SHA256 hex）
- canonical 串：`dealerId={dealerId}&ts={ts}&nonce={nonce}`
- 有效期：10 分钟；过期/签名不匹配 → H5 不允许下单

### 2.3 订单归属（健行天下购买订单）

```

┌──────────────────────────────────────────────────────────────────────────┐

订单详情（只读为主）：
基础信息 | 支付记录 | 退款记录 | 归属信息 | 操作日志
```

### 2.4 结算记录（对账/汇总）

```

┌──────────────────────────────────────────────────────────────────────────┐
│ 结算单号 | 周期 | 订单数 | 应结算金额 | 状态(待确认/已结算/冻结) | 操作[查看明细/导出] │
└──────────────────────────────────────────────────────────────────────────┘

明细：
订单号 | 金额 | 手续费 | 结算金额 | 支付时间
```
